# python-unittest-generator
Auto generating unit testing file and description for python class

******

#### schema format

```
name: "name of module"
path: "path for module"
tests:
    "name of tested method":
        test_instances:
            - input: [input1, input2, ...]
            - output: 'output'
    ...
```


******

#### How to use

python unittest-generator.py
